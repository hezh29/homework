## 2.

```
list = {60,65,75,80,90,95}
FOR i from list.length - 1 to 0 DO
    IF insertValue < list[i] THEN
        list[i+1] = list[i]
    ELSE
        list[i+1] = insertValue
        END FOR
    END IF
END FOR
IF insertValue < list[0] THEN
    list[0] = insertValue
END IF
```
